# Range Queries
>
Range queries are those which retrieve data from a fixed range from an array. This can include finding the sum of all elements in the range, the maximum element and the minimum element.

```ad-important
The following range query techniques are made to be efficient for all possible ranges in an array. Therefore, if you would like to do range queries on a range the size of the array (i.e find the sum of an array, find the maximum or minimum value), then you will be better off using a [[Binary search tree]] instead.
```
